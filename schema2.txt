Supabase Database Architecture Documentation
This document describes the database schema, relationships, and Row Level Security (RLS)
configuration for the healthcare appointment system implemented using Supabase. The database
supports roles for patients, doctors, and administrators while enforcing strict access control through
RLS policies.
